Concepts clés

Partie 1 - Découvrez le web et ses origines

Web

Le Web est une toile d'araignée, avec ses nombreux fils connectés, le web est un ensemble de pages, accessibles depuis un lien hypertexte. Le Web fut créé par Tim Berners-Lee.

Internet

Internet est un réseau qui permet de communiquer entre ordinateurs. Internet a été inventé avant le web.

Cloud

Le cloud permet de stocker des fichiers, des données et des services à travers une interface web.

W3C

Le W3C est un organisme qui a pris le relais de Tim Berners-Lee pour faire évoluer les technologies du web.

Partie 2 - Distinguez les langages du Web

Langages client

Les langages client servent à fournir des informations de base pour la construction des sites web. Les langages client décrivent comment le site web doit s’afficher. Ces langages sont également appelés des langages Frontend. Exemples de langages : HTML, CSS, JavaScript.

Navigateurs

Le rôle des navigateurs est de traduire les langages HTML, CSS et JavaScript sous la forme de sites web utilisables par tout le monde. Exemples de navigateurs : Google Chrome, Mozilla Firefox, Edge...

Langages serveur

Les langages serveur décrivent comment le site web doit se comporter. Ces langages sont également appelés des langages Backend. Exemples de langages : PHP, Java, Python...

Frameworks

Les Frameworks sont des structures entières de code écrits par d'autres développeurs qui sont réutilisables pour gagner du temps. Exemples de Frameworks Backend : Symfony et Zend pour PHP, Java EE pour Java...

Bases de données

Les bases de données sont des logiciels dédiés au stockage de données. Le langage utilisé pour les bases de données relationnelles est le langage SQL. Exemples de bases de données : MySQL, Postgre SQL, SQL Server... Il existe également des bases de données non relationnelles comme MongoDB, Redis, Neo4j... qui eux utilise un autre langage.

Site responsive

Un site responsive est un site web qui s'adapte automatiquement à la petite taille des smartphones. Il s'utilise avec les langages Frontend.

Application mobile

Une application mobile est une application conçue spécialement et uniquement pour un smartphone ou tablette. Ces applications sont appelées des applications "native". Elle s'utilise avec des langages mobiles Android ou IOS (Kotlin, Java ou Swift)

Partie 3 - Découvrez le fonctionnement du réseau

Serveurs

Les serveurs sont les machines qui possèdent les sites web et qui les distribuent aux clients. Les serveurs sont stockés dans des datacenters.

Adresse IP

Chaque ordinateur possède une adresse IP. Il s'agit d'une suite de nombres. Exemple : 205.89.177.26.

Nom d’hôte et DNS

Le nom d'hôte est le nom du service que l'on veut contacter. Exemple : www.google.com. Le DNS est un annuaire qui permet de traduire le nom d'hôte en adresse IP.

Protocoles

Les protocoles sont des langages de communication pour que les machines se parlent entre elles. Il existe deux niveaux de protocoles : les protocoles bas niveau (TCP, UDP) et les protocoles haut niveau (HTTP,FTP...).

Partie 4 - Identifiez les métiers du web

Développeurs

Les développeurs sont des personnes qui conçoivent, testent, documentent et maintiennent des applications, des sites Internet ou des logiciels. Les développeurs utilisent des langages de programmation pour coder un site ou une application. De nombreuses spécialités existent (Frontend, backend, Full Stack, mobile).

Product Managers et chefs de projets

Le Product Manager assure que le produit évolue dans la bonne direction, en anticipant les besoins futurs et en définissant une stratégie à long terme. Le chef de projet pilote la réalisation du produit, de l'élaboration jusqu'à son lancement.

Designers

Il existe deux types de designers. L'UI designer qui va s'occuper de l'interface du site web (mixe les éléments graphiques et fonctionnels pour rendre le site attractif, agréable et accessible) et l'UX designer qui va s'occuper de l'expérience globale de l'utilisateur (crée des prototypes, test si le produit répond aux attentes...).

Testeurs

Le testeur logiciel ou ingénieur QA est chargé de concevoir, d'exécuter et d'automatiser les tests pour garantir la qualité des logiciels ou des produits. Il identifie les bugs et les problèmes de performance afin de les résoudre avant la mise en production.

Techniciens informatique

Le technicien informatique a un rôle polyvalent. Il s'occupe de la gestion du parc informatique, du support technique et de l'accès aux services réseaux des entreprises.

Langages de développement web

HTML

Front-End

Squelette du site Web. Il définit la structure de base du contenu, comme les titres, les paragraphes, les images, etc.

CSS

Front-End

Il définit le visuel du site, comme les couleurs, la police, les tailles, etc.

JavaScript

Front-End

Il ajoute de l'interaction sur le site, comme des menus déroulants, un affichage de message lors de la validation de formulaires, etc.

JavaScript

Back-End

Il permet la mise en place, la configuration et la maintenance du site. Il utilise Node.js qui est une plateforme logicielle libre. Cela permet à JavaScript de s'exécuter en tant que langage serveur.

PHP

Back-End

C'est un langage de programmation utilisé pour créer des pages web dynamiques.

Java

Back-End

C'est un langage multiplateformes qui fonctionne sur tous les appareils.

Python

Back-End

C'est un langage multiplateformes et open-source.

C#

Back-End

C'est un langage destiné à formuler des algorithmes et produire des programmes informatiques qui les appliquent.

SQL

Bases de données

Il permet d'exploiter des bases de données relationnelles. Il sert à rechercher, à ajouter, à modifier ou à supprimer des données dans les bases de données.

Boîte à outils du développeur

  • IDE comme VS Code : Les IDE sont des boîtes à outils, permettant d’écrire du code, mais aussi de gérer d’autres aspects du développement. Ils permettent de retrouver facilement l’arborescence des fichiers, d’avoir une complétion automatique du code, de visualiser le code avec une coloration, etc.
  • Github : GitHub est une plateforme basée sur le cloud où vous pouvez stocker, partager et travailler avec d'autres pour écrire du code. Github intéragit avec Git. Git est un logiciel de gestion de versions décentralisé. Un gestionnaire de versions ("appelé versioning" en anglais) est un programme qui permet aux développeurs de conserver un historique des modifications et des versions de tous les fichiers. Github permet donc de conserver sur internet ou sur un réseau, les fichiers enregistrés sur Git.
  • Librairies / bibliothèques : Les librairies/bibliothèques sont des morceaux de code écrits par d'autres développeurs qui sont réutilisables pour gagner du temps.
  • Frameworks : Les Frameworks sont des structures entières de code écrits par d'autres développeurs qui sont réutilisables pour gagner du temps. Exemples de Frameworks Backend : Symfony et Zend pour PHP, Java EE pour Java, Django pour Python...
